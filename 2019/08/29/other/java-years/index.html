    我学java这么多年

    谢晋杰 其他
    创建时间:2019-08-29 21:37
    字数:977 阅读:
    评论:
    1. 迷失
    2. 悔悟

    对于一个物理科班出身的程序员，认为程序如同物理世界是规律的、秩序的。

    大学才接触到编程，之前都不知道这回事儿。大一开的C语言也算是一次尝试编程，能敲出一个hello word就觉得兴奋不已，跑到机房敲demo感觉时间过得很快，也很满足。C课程结束之后就没有了。。。

    到现在为止，谈到物理内心依然充满火焰，但是当初科班的物理生涯却没有带来太多满足感，反而编程带来不少快乐。大三开始考虑之后的工作，想试试编程，咨询了一下在做java的大表哥，就认准方向。那时候天真的认为会java就可以做java了，就买了一本head first java开始啃，以为自己啃完了就入门了。书中什么对象、异常、多线程、网络，对着实例敲代码不亦乐乎。当翻完最后一页的时候，我开始迷惘了，我他么能做什么，有一种感觉自己会java又什么都不会做的感觉。不过要感谢这本书让我很开心的进入了java新手村。

    入坑。后来找到了传智播客韩顺平的一套java视频课程，就开始撸视频，撸代码。跟着视频做了一个坦克大战游戏，学生管理系统，给女友秀了一把。可是感觉离开视频教程自己还是一个渣渣。然后自己查资料鼓捣出来了个俄罗斯方块的游戏。看了一下招聘，发现招聘java还需要会html/css和sql，于是乎，便手撸了head first html/css和head first sql。也算是head first爱好者了。

    大四找工作磕磕绊绊有幸能够进入一家大型国企。签了工作之后，听hr说了linux要学好。linux也是第一次听说，本着没事找事的理念，于是乎就开始给我的电脑安装arch linux，整了一个月arch linux安好，电脑完蛋了。拿给女友看，女友一脸懵逼看着我只有命令行的电脑。

    到了公司开了眼见，web开发，大数据，中间件。好在一进公司是从装系统开始培训，就开始了白天装系统，晚上看视频的节奏。感觉自己基础不够，空闲时间又撸了一本书叫java核心技术，撸玩卷一之后，发现这货就是java api文档+demo，不过感觉功力有些许长进。在公司接触学习spring等框架，也接触到了大数据和数据库相关的知识，收益匪浅。

    迷失

    工作一年多后离职进京之际，在一场开发者大会上，一位年轻的讲师Delton Ding讲述着自己如何在ruby代码中混合汇编语言解决问题，如何用人工智能合成音乐，这着实诧异了我，程序员原来能有如此境界。后来我才知道，他的编程能力是童子功。然后自己想想是不是可以往底层钻一钻，于是开始撸算法第四版，这一次我发现自己撸不动了，只能撸到第三章，感觉自己底子太薄了，于是乎就报了计算机第二学位的考试。考试没通过，却把英语知识恶补了一下。期间买了服务器，搭了博客，也给女友做了一个web应用，还考了驾照。

    悔悟

    考试失利后，进入了一家大型的外包公司继续做开发，兜兜转转一圈后，发现自己对前沿的技术框架学习还不是精通且深入，就去学习底层知识有点走火入魔，但这一遭也让我更明确了方向。
